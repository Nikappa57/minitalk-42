### Installing

Clone the repository:

```
git clone https://github.com/Nikappa57/minitalk-42.git
cd minitalk-42
```

Compilation:

```
make all
make bonus
```

## Usage <a name="usage"></a>

### Server

```
./server
./server_bonus
```

### Client

```
./client [SERVER_PID] [MESSAGE]
./client_bonus [SERVER_PID] [MESSAGE]
```
